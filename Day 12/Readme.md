# Country Search App

Overview

This is a simple web application that allows users to search for countries using an external API.

Features

- User input search
- Fetch data from REST Countries API
- Display country name, population, and flag

How It Works

1. User types a country name
2. Clicks the search button
3. JavaScript sends a request using "fetch()"
4. Data is received in JSON format
5. Results are rendered dynamically on the page

Concepts Practiced

- DOM manipulation
- Async/Await
- API integration
- Dynamic rendering

Notes

This project focuses on connecting frontend logic with external data sources.
